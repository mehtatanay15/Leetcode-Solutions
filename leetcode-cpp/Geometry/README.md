# Geometry Problems

Geometry-based problems involving coordinate systems, shapes, and spatial relationships.

## Problems Solved

| # | Problem | Approach | Time | Space |
|---|---------|----------|------|-------|
| 3464 | [Maximize Distance From Origin on Square Perimeter](problem_3464.cpp) | Binary Search + Greedy on Unrolled Perimeter | O(n² * log(n) * log(side)) | O(n) |

## Subcategories
- **SquarePerimeter/**: Problems involving points on square perimeter
- **Optimization/**: Geometric optimization problems

## Key Techniques
- Perimeter unrolling (convert 2D problem to 1D)
- Binary search on answer
- Greedy validation with geometric constraints
