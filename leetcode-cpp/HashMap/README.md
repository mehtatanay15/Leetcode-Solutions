# HashMap Problems

Hash map and frequency-based problems for efficient data lookup and counting.

## Problems Solved

| # | Problem | Approach | Time | Space |
|---|---------|----------|------|-------|
| 2784 | [Check if it is Possible to Make Array Equal](Frequency/problem_2784.cpp) | Frequency Counting + Validation | O(n) | O(n) |
| 3280 | [Minimum Mirror Pair Distance](Other/problem_3280.cpp) | HashMap + Number Reversal | O(n*d) | O(n) |
| 3488 | [Minimum Circular Distance Between Elements With Same Value](Other/problem_3488.cpp) | HashMap + Binary Search | O(m log m + n log m) | O(m) |

## Subcategories
- **Frequency/**: Frequency counting and validation
- **Other/**: General HashMap-based problems
- **Grouping/**: Grouping elements by value
- **TwoSum/**: Two sum variants
- **LongestSubstring/**: Longest substring problems
- **SubarraySums/**: Subarray sum problems
