# Tree Problems

Binary tree problems including traversal (DFS/BFS), comparison, and path analysis.

## Problems Solved

| # | Problem | Approach | Time | Space |
|---|---------|----------|------|-------|
| 100 | [Same Tree](Other/problem_100.cpp) | Recursive DFS | O(min(m,n)) | O(min(h1,h2)) |
| 101 | [Symmetric Tree](Other/problem_101.cpp) | Recursive DFS (Mirror Check) | O(n) | O(h) |
| 543 | [Diameter of Binary Tree](Other/problem_543.cpp) | DFS Post-order + Height | O(n) | O(h) |
| 2196 | [Create Binary Tree From Descriptions](Other/problem_2196.cpp) | Hashmap + Set for Root Identification | O(n) | O(n) |

## Subcategories
- **DFS-Inorder/**: Inorder traversal
- **DFS-Preorder/**: Preorder traversal
- **DFS-Postorder/**: Postorder traversal
- **BFS-LevelOrder/**: Level-order traversal
- **BST/**: Binary search tree problems
- **PathSum/**: Path sum problems
- **LCA/**: Lowest common ancestor
- **Serialization/**: Tree serialization
- **Other/**: Miscellaneous tree problems
